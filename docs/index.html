YACV

Yet Another Csv Viewer

A simple - recursive descent - parser for CSV files. The grammar is based on rfc4180 and the parser is just a couple of hours of work, be kind! Default delimiter is ',' but you can choose between a few different delimiters. For example, when exporting a csv with MS Excel in italian the delimiter is a ';'.


If you have any suggestion or you want to file a bug, please open an issue here.
